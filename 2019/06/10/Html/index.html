    表单域（from）

    from标签

    常用属性：

    1. Action
      在表单收集到信息后，需要将信息传递给服务器进行处理，action属性用于指定接收并处理表单数据的服务器程序的url地址。
    2. method
      用于设置表单数据的提交方式，其取值为get或post。
    3. name
      用于指定表单的名称，以区分同一个页面中的多个表单。

    注意： 每个表单都应该有自己表单域。
