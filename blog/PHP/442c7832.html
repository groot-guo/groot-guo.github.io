变量

定义变量名的方法

  • 必须以字母或者下划线 “_” 开头
  • 只能由数字或者字母，以及 “_” 组成，换可以加汉字
  • 不允许有空格，多个单词建议 驼峰

变量的数据类型

可以通过 memory_get_usage来判断不同数据消耗的内存

  • int 整型

    • php中会将其他进制的数自动转换成十进制值输出

  • bool 布尔值

  • string 字符串

    • 可以使用 ''或者""包含

    • 单双引号可以互相嵌入

    • 多行字符串时（$变量 = <<<GOD GOD;）

  • float 浮点型

    • 可以使用 e 或者 E来表示小数的科学计数法
